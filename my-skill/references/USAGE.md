# 使用说明

## 你给我的输入（推荐）
- 【待分析句子】一整句英文
- 【上下文】前后各1句（可选，但强烈建议）
- 【关注点】例如：从句关系 / 非谓语 / 指代 / 翻译更顺 / 难词

如果你怀疑有“熟词偏译”，也可以在关注点里写：熟词偏译，并标出你觉得别扭的词（如 address/issue/subject/claim 等）。

## 我给你的输出
按 [../SKILL.md](../SKILL.md) 的模板，固定包含：
- 主干
- 分层结构
- 语法难点
- 难词/关键词语境义与翻译策略（含：常见义→语境义、触发线索、与常见义的关系）
- 意群对照翻译
- 整句：直译 / 意译 / 背诵版

## 小技巧
- 如果你在做真题阅读，建议把“题干+定位句+前后句”一起给我，难词解释会更准。
